# Customer Churn Prediction using ANN

This project leverages an **Artificial Neural Network (ANN)** to predict customer churn. The model is trained on customer data to identify factors that influence churn and provides actionable insights for businesses.

## Features
- Data Preprocessing and Cleaning
- Exploratory Data Analysis (EDA)
- ANN Model Implementation using TensorFlow/Keras
- Model Training, Tuning, and Evaluation
- Predictions and Insights

## Technologies Used
- Python
- Pandas, NumPy, Matplotlib, Seaborn
- TensorFlow / Keras
- Scikit-learn
- Jupyter Notebook
